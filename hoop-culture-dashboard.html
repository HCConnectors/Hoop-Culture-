🚀 DEPLOY TO GITHUB PAGES

Free hosting at yourusername.github.io/hoop-culture — live in under 5 minutes. No credit card, no domain needed.

STEP 1 — EXPORT YOUR DATA

Click ⬇ EXPORT in the top bar first. This downloads a hoop-culture-backup.json file — keep this safe, it's your data.

STEP 2 — CREATE GITHUB ACCOUNT

Go to github.com → Sign up free. Pick a username like hoopculture — it becomes part of your URL.

STEP 3 — CREATE REPOSITORY

Click New repository → Name it hoop-culture → Set to Public → Click Create repository.

STEP 4 — UPLOAD THE FILE

In your new repo click uploading an existing file → drag in the hoop-culture-dashboard.html file → Rename it to index.html → Click Commit changes.

STEP 5 — ENABLE GITHUB PAGES

Go to repo Settings → scroll to Pages → under Source select Deploy from branch → choose main → click Save. Your URL appears at the top in ~60 seconds.

⚡ IMPORTANT — DATA SAVES PER BROWSER

Your edits save automatically in whatever browser you use. To move data between devices or browsers: EXPORT on the old device → open the live URL on the new device → IMPORT the file. Done.
